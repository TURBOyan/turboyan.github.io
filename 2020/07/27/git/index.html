GitHub提供了两种管理代码方式，一种是命令行形式（GitBash），另一种是窗口界面（GitGUI）形式。

GUI管理的好处是方便上手，处理迅速，但是在这里我还是建议新手使用命令行形式，因为那样不仅可以更深入的了解到Git的管理代码形式，还能使用到Git的更多GUI无法实现的功能。
